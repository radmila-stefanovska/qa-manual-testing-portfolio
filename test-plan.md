# Test Plan

## Project

SauceDemo Web Application

https://www.saucedemo.com/

---

## Objective

The purpose of this test plan is to verify that the SauceDemo web application functions correctly and provides a reliable user experience.

---

## Scope

The following modules will be tested:

- Login
- Products
- Shopping Cart
- Checkout
- Logout

---

## Testing Types

- Functional Testing
- Smoke Testing
- Regression Testing
- UI Testing
- Negative Testing

---

## Test Environment

Operating System:
Windows 11

Browser:
Google Chrome

Application URL:
https://www.saucedemo.com/

---

## Entry Criteria

- Application is accessible
- Test environment is available
- Test data is prepared

---

## Exit Criteria

- All High Priority test cases executed
- Critical defects resolved
- Smoke testing completed successfully

---

## Risks

- Browser compatibility issues
- Internet connection problems
- Test environment instability

---

## Deliverables

- Test Plan
- Test Cases
- Bug Reports
- Screenshots
- Test Execution Report
